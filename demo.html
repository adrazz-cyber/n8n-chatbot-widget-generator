N8N Chatbot Demo Page

This is a demonstration page showing how the N8N chatbot widget appears on a website.

The chat button should appear in the bottom-right corner of this page. Click it to test the chat functionality.

Note: Make sure to replace the webhook URL in the embed code with your actual N8N workflow webhook URL for the chat to work properly.

How to Use

  1. Generate your embed code using the Widget Generator
  2. Replace the webhook URL with your N8N webhook
  3. Paste the code into your website's HTML
  4. Test the chat functionality

Current Configuration

This demo uses the default BlueChilli configuration. You can customise:

  • Button colours and size
  • Chat window appearance
  • Message styling
  • Company branding
  • Welcome messages
